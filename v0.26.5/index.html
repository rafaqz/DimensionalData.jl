Intelligent indexing

DimensionalData.jl provides no-cost abstractions for named indexing, and fast index lookups.

Powerful Array manipulation

broadcast, reduce, permutedims, and groupby operations.

Seamlessly integrated with the julia ecosystem

Works with most methods that accept a regular `Array`. If a method accepts numeric indices or dims=X in base, you should be able to use DimensionalData.jl dims.
